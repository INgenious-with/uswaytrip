<h2>5. 느낀점</h2>
<h3>5-1. 어려웠던 부분</h3>

📌Spring MVC 구조에 대한 이해 부족<br>
✔ Spring MVC 구조에 대한 이해도가 부족하여 기능 구현에 어려움을 겪었습니다. 모르는 부분은 팀원들과 소통하며 학습하고,<br>그 덕분에 맡은 기능을 구현할 수 있었습니다.
<br><br>
📌Bootstrap을 통한 화면 구성<br>
✔ 프로젝트에 적합한 Bootstrap 템플릿을 찾아 적용했으나, 모든 CSS와 JS 관련 사항을 파악하는 데 어려움이 있었습니다. <br>하지만 프로젝트에서 필요하지 않은 부분을 제외하고, 필요한 기능만을 잘 추려 커스터마이징 하면서 문제를 극복할 수 있었습니다. 
<br><br>

<h3>5-2. 총평</h3>
<strong>🐢천천히 가야한다.</strong><br><br>
✨스프링 프로젝트를 진행하면서 느낀 점은, Java와 웹 개발에 대한 기본 지식이 부족했기 때문에 많은 어려움이 있었다는 것입니다.<br>
국비 과정에서 처음 웹 개발을 접했고, Java는 물론 HTML, CSS, JavaScript 등 모든 기술이 처음이었기에 많은 부분에서 미숙했습니다.<br>
그렇게 빠르게 지나간 3개월 동안 맡은 기능을 제대로 구현하기에는 역량이 부족했지만, 수업 내용을 반복하며 부족한 부분을 채워갔습니다.<br>
그 결과, 제 역할을 잘 수행할 수 있었습니다.<br>
"첫술에 배부를까?"라는 속담처럼 모든 일은 처음부터 완벽할 수 없으며, 개발도 마찬가지라는 생각이 듭니다.<br>
제 부족한 점을 정확히 알고, 앞으로 더 나아가며 계속 성장하겠습니다. 🥪
